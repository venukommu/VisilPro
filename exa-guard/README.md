# Exa Guard

AI-Enabled Live Proctoring for Online Examinations.

## Overview
Exa Guard is an AI-driven proctoring system integrating CCTV and laptop cameras with audio analytics to detect malpractices, ensure fairness, and uphold the credibility of online assessments.

## Features
- **AI-Powered Multi-Modal Monitoring**: Integrate CCTV and laptop camera feeds.
- **Anomaly Detection**: Identify multiple faces, absence of candidates, etc.
- **Audio Analytics**: Detect background speech.
- **Real-Time Alerts**: Notifications for proctors.

## Tech Stack
- Java 17
- Spring Boot 3.1.6
- H2 Database
- Spring WebSocket

## Running the Application
1. Ensure Java 17 and Maven are installed.
2. Run `mvn spring-boot:run`.
